1.以 .tar 结尾的归档是否已被压缩？

B. 不是

2.bzip2 压缩相对于 gzip 压缩的优势是什么？

A. 压缩率更高
B. 压缩更快
C. bzip2 是默认安装程序，而 gzip 不是默认安装的

3.Telnet 协议的主要缺陷是什么？

A. 不需要输入密码
B. 数据没有被加密
C. 速度慢

4.当用 ssh-keygen 生成一对公私密钥对时，输入密码短语（Passphrase）的作用是什么？

A. 加密私钥
B. 加密公钥
C. 加密私钥和公钥

5.SSH 程序的主要作用是什么？

A. 监控网络活动，以防黑客攻击
B. 记录并分析服务器指纹，并进行对比
C. 将解密的私钥加载入内存，以便之后不需要再每次输入密码短语（Passphrase）

6.使用 scp 将位于家目录的一张图片 photo.jpg 拷贝到远程电脑的 18007 端口上，该怎么做呢？

A. scp -p 18007 photo.jpg login@ip:copy_photo.jpg
B. scp -P 18007 login@ip:copy_photo.jpg photo.jpg
C. scp -P 18007 photo.jpg login@ip:copy_photo.jpg
D. scp -oPort=18007 login@ip:copy_photo.jpg photo.jpg

7.在 LISTEN 状态的网络连接是否正在进行数据交换？

A. 是
B. 不是

8.iptables 中的 OUTPUT 部分的 source 对应什么？

A. 网络接入服务商
B. 本地电脑
C. 远端电脑

9.如果要创建一个 iptables 规则，用来允许 21 端口（ftp 端口）用 TCP 协议来向外传输数据，该怎么做？

A. iptables -A OUTPUT -p tcp --dport 21 -j ACCEPT
B. iptables -A INPUT -p tcp --dport ftp -j ACCEPT
C. iptables -A OUTPUT -p tcp --dport ftp -j DROP
D. iptables -A INPUT -p icmp --dport 21 -j ACCEPT

10.一般我们依次用哪几个命令来实现从软件的源码编译安装?

A. ./configure -> make -> make install
B. ./prepare -> compile -> install
C. make configure -> make -> make install

  1. B
  2. A
  3. B
  4. A
  5. C
  6. C
  7. B
  8. B
  9. A
